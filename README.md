
# My Portofolio

A basic Home page made with react which supports Google Speedshet and Api Key Blogger V3.


## Features

- Home Page
- Api Key BloggerV3
- Google Speedshet



## Run Locally

Clone the project

```bash
  git clone https://github.com/appank/Portofolio.git
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

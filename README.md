
# My Portofolio

A basic Home page made with react which supports  Api Key Blogger V3.


## Features

- CSS
- Api Key BloggerV3



## Run Locally

Clone the project

```bash
  git clone https://github.com/appank/Portofolio.git
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
